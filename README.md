## 采用 Nuxt3 编写的练手博客,目前正在开发中。。

使用了以下技术

- Nuxt3
- Vite
- Vue3
- Tailwindcss
- Pinia
- UmiRequest

# 使用

```sh
pnpm i
pnpm dev
```
